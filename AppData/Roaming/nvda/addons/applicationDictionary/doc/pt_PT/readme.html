Apresentação

Este extra permite criar dicionários do NVDA específicos para uma aplicação. Para utilizá-lo, basta na aplicação, para a qual deseja criar um dicionário, pressionar NVDA+control+shift+d. O diálogo de dicionários do NVDA irá abrir para que se possa adicionar entradas válidas somente para a aplicação ativa (o nome da aplicação deverá aparecer no título do diálogo). O comando padrão pode ser modificado através do diálogo "Definir comandos:" na secção "Configuração", opção "Apresenta o diálogo do dicionário de aplicações".

Comando

NVDA+Control+Shift+d
