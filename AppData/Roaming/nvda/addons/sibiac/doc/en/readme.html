Single Image Blob Interface Accessible Control

  • Authors: Alexey Zhelezov, www.azslow.com

This add-on provides accessible overlays for graphic interfaces

IMPORTANT: please also install sibiac_ocr add-on, it contains required OCR libaray

All keybord shortcuts are provided by SIBIAC itself, so usual NVDA rules and settings are ignored.

  • tab and shift+tab switch between virtual controls
  • arrows change the selection within one control or values
  • enter key perform control specific operation

The following overlays are included into this version. Please check my forum for details

  • Behringer X-Touch Editor. Allows change device mode and load layouts from files into hardware. Layout editing is not supported. On loading and uploading the editor asks for confirmation in pup-up dialog. The dialog is not accessible, but should be announced by NVDA. Press enter to confirm and return to the editor.
  • REAPER:
    • XLN Addictive Drums 2. Everything except MIDI and details in FXes tuning
    • GTune. Alll operations.
    • XLN Addictive Keys. Preset selection only
    • NI Guitar Rig 5. Preset selection only
    • NI Absynth. Preset selection only
    • Cakewalk instruments. Preset selection. In the browser, loading preset with Alt+Enter.
    • EZ Mix 2. Complete
  • Melodyne
